📄 КОНЦЕПЦИЯ ПРОЕКТА
Платформа для роупджампинг-сообщества
1. 🧠 Идея проекта

Создать платформу (веб-сайт), которая объединяет:

команды по роупджампингу
участников (новичков и опытных)
мероприятия
объекты
медиа-контент

Цель:
Сделать единое место, где можно:

находить прыжки
организовывать мероприятия
общаться
делиться опытом
сохранять историю участия
2. 🎯 Ценность продукта
Для участников
удобный поиск мероприятий
возможность участвовать в прыжках
профиль с историей
общение с комьюнити
доступ к проверенным командам
Для команд
инструмент для анонсов
сбор заявок
привлечение аудитории
публикация контента
формирование репутации
Для новичков
понятное объяснение
безопасный вход в тему
выбор команды
информация о первом прыжке

2. 🧩 Лента (Feed)
🎯 Главная задача ленты
показать, что здесь есть жизнь;
объединить людей;
дать место для общения и контента;
продвигать мероприятия и команды.

🧠 Принцип ленты

Лента должна быть:

простой;
визуальной (фото/видео);
не перегруженной;
с понятной логикой.

👉 Лучше простая лента, чем “умный алгоритм”.

Очень важно сразу определить типы.

🧠 Принципы системы бейджей
Никакого ручного ввода прыжков
Всё считается через мероприятия
Минимум действий от организатора (только “Состоялось”)
Бейджи = история и опыт, а не “кто круче”
Часть бейджей — авто, часть — вручную (редкие)
🧩 1. Бейджи за участие (основа)
🎯 Старт
Первый прыжок
→ участие в первом мероприятии
Вернулся снова
→ 2 мероприятия
🔁 Прогресс
5 мероприятий → “В теме”
10 мероприятий → “Свой человек”
25 мероприятий → “Постоянный участник”
50 мероприятий → “Живёт этим”
100 мероприятий → “Ядро сообщества”

👉 считаются автоматически

🗺 2. Бейджи за объекты (ключевая фишка 🔥)
📍 Уникальные объекты
Первый объект → “Новый горизонт”
3 объекта → “Исследователь”
5 объектов → “Открывает новые места”
10 объектов → “Путешественник”
20 объектов → “Коллекционер объектов”
🏗 Бейдж конкретного объекта

Каждый объект даёт свой бейдж:

“Северный мост”
“Башня X”

👉 формирует “карту опыта”

🧗 3. Бейджи за высоту (очень важные)

Считаются по максимальной высоте объекта, где был пользователь.

Уровни:
30+ м → “Первый уровень”
50+ м → “Выше страха”
100+ м → “Сотка”
150+ м → “Граница”
200+ м → “Высота”
300+ м → “Свободное падение”

👉 выдаются автоматически после события

🎪 4. Бейджи событий
🏁 Уникальные события
Фестиваль
Winter Jump
Night Jump

👉 выдаются, если событие помечено организатором
